# Playwright Automation Project - SauceDemo
--- Overview---
This project automates test scenarios for the SauceDemo website using Playwright with JavaScript following the Page Object Model (POM) design pattern.

Website: https://www.saucedemo.com/

---
 Tools & Technologies used
- Playwright
- JavaScript
- Node.js
- Page Object Model (POM)
- Git & GitHub

---Automated Scenarios
Scenario 1: Login with Valid Credentials
- User enters valid username and password
- User clicks login
- User is redirected to Inventory Page

---Scenario 2: Place Order Successfully
- User logs in
- Adds product to cart
- Proceeds to checkout
- Fills checkout details
- Completes order
- Verifies "Thank you for your order!" message

---
